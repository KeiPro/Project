1. 클래스의 멤버 함수에서 다른 클래스의 멤버 변수나 멤버 함수를 사용하려는 경우.

예를들어 class Player{}; 클래스가 있고 class GameManager{}; 클래스가 있다.
이를 순서대로
class Player{

};

class GameManager{

};

이렇게 선언되어 있을 경우 Player의 멤버 함수에서 GameManager의 멤버 변수나 함수를 사용하지 못한다.
지금 이해한 바로는 저 순서로 되어 있던, 두 클래스가 다른 .h파일에 선언되어 있던 자신의 클래스 안에 있는
멤버 함수는 다른 클래스에 대한 내용을 알고있어야 되므로 상단에 접근하고 싶은 클래스의 이름을 표현해줘야 한다.
따라서 class Player{}; 클래스 위에 class GameManager;라고 선언을 해주어야 이용이 가능하다.
